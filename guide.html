03

⏱ 約 5 分鐘

Claude 桌面應用程式整合了聊天介面與 Claude Code 終端機，是新手最簡單的入門方式，不需要安裝 Node.js 或 Git，直接下載安裝即可。

桌面應用程式已內建所有必要環境。不需要 Node.js、不需要 Git、不需要 PowerShell 指令。

06

告訴 AI 你的工作背景，之後不用每次重複說明
⏱ 約 5 分鐘
📌
CLAUDE.md 是什麼？

放在工作資料夾裡的文字檔，Claude Code 每次啟動都會自動讀取。把它想像成「給 AI 的工作說明書」—— 設定好後，AI 一開始就知道你是消防製圖人員、要審查哪些項目、用什麼格式輸出。

步驟 6-1｜用記事本建立檔案

按 Windows 鍵 搜尋「記事本」，開啟。

先不要輸入任何內容，按 Ctrl + S 存檔：
存檔位置選 C:\消防審圖，檔案名稱輸入 CLAUDE.md，
存檔類型務必選 「所有檔案 (*.*)」（否則會存成 CLAUDE.md.txt），按儲存。

步驟 6-2｜貼入以下內容

將下方整個程式碼區塊的內容全選複製（Ctrl+A 全選，Ctrl+C 複製），貼到記事本中：

CLAUDE.md — 消防審圖設定範本（可依需求修改）
# 消防圖面輔助審查工作環境設定

## 工作背景
- 我是消防設備製圖人員，負責消防安全設備圖面的繪製與初步審查
- 主要使用 AutoCAD，圖面格式包含 DXF、PDF 及圖片（JPG/PNG）
- 適用法規：《各類場所消防安全設備設置標準》（台灣）

## 主要審查項目
- 自動撒水設備：灑水頭間距、保護面積、水平距離
- 火警自動警報設備：探測器間距、覆蓋半徑、類型選用
- 緊急廣播設備：揚聲器配置、音壓規定
- 消防栓：位置、有效水平距離
- 避難逃生：走廊寬度、出口數量、疏散路徑

## 回答格式要求
- 請用繁體中文回答所有問題
- 發現疑義時，請同時引用設置標準的條文編號
- 審查結果請以「符合」「不符合」「需確認」三種狀態標示
- 數值計算需顯示計算過程，方便核對
- 輸出審查清單時，請使用表格格式

## 資料夾說明
- 圖面輸入/：待審查的 AutoCAD 圖面及相關文件
- 審查報告輸出/：存放 AI 輔助生成的審查意見草稿
- 參考資料/：法規條文摘要、常見缺失範本等

## 重要提醒
- AI 審查結果僅供輔助參考，最終判斷仍需具資格人員確認
- 如遇模糊地帶，請說明需要進一步核對的項目

遇到問題怎麼辦？
常見狀況與對應處理方式

❌ 安裝後找不到 Claude 應用程式圖示

  • 1
    按 Windows 鍵搜尋「Claude」，找到後可右鍵釘選到桌面
  • 2
    找不到：重新啟動電腦後再搜尋
  • 3
    仍無效：至 claude.com/download 重新下載安裝

❌ 開啟 App 後登入頁面白畫面或無法載入

  • 1
    確認網路正常（試開 google.com）
  • 2
    工作列右下角 Claude 圖示按右鍵 →「結束」，重新開啟
  • 3
    公司電腦：確認 claude.ai 與 api.anthropic.com 未被防火牆封鎖

❌ 終端機輸入 claude 顯示「找不到命令」

  • 1
    改用桌面 App 內建的 Claude Code 終端機，不需額外設定
  • 2
    若要用外部 cmd：完全關閉視窗，重新開啟再試
  • 3
    仍無效：重新啟動電腦讓 PATH 環境變數生效

❌ 顯示「Authentication failed」或「Unauthorized」

登入狀態已過期。輸入 /logout，再輸入 claude 重新登入。或在 App 設定中重新登入帳號。

❌ 顯示「Insufficient credits」或「Payment required」

訂閱有問題。登入 claude.ai，確認 Pro 訂閱狀態，或確認信用卡有正常扣款。

⚠️ AI 回覆的中文出現亂碼

桌面 App 內建終端機通常不會有此問題。若用外部 cmd，在啟動前輸入 chcp 65001。建議改用 App 內建終端機，或安裝 Microsoft Store 免費的 Windows Terminal。

⚠️ 如何更新 Claude 到最新版本

桌面應用程式會自動在背景更新，通常不需要手動處理。若想立即更新，在 Claude App 選單中找「Check for updates」，或在終端機輸入 claude update。
