## 🚒 AI 能幫你做什麼？

- **圖面初步審查** — 提供 DXF 或圖面截圖，AI 說明設備配置是否符合設置標準常見規定
- **間距與覆蓋計算** — 驗算灑水頭間距、探測器覆蓋半徑，並顯示計算過程
- **缺漏清單自動整理** — 比對圖說常見缺失，產生結構化待補項目清單
- **法規條文對照查詢** — 用中文問問題，快速找到設置標準相關條文
- **批次圖面彙整** — 一次處理多份案件，整合審查意見並輸出統一格式報告

> ⚠️ **重要：** AI 審查結果僅供輔助確認與提示，最終審查判斷仍需由具備資格的消防設備師（士）負責。
