## AI 可協助的工作項目

- **圖面初步審查** — 讀取 PDF、截圖，檢查設備配置是否有明顯疑點
- **座標與間距計算** — 讀取 DXF，驗算撒水頭間距、探測器配置，並列出計算過程
- **缺失清單整理** — 對照常見缺失清單，產生結構化待確認項目
- **法規條文對照** — 依據提供的參考資料查對條文，無法確認時標示需人工查證
- **審查報告輸出** — 整合分析結果，輸出 Markdown 格式審查意見草稿

**重要：** AI 審查結果僅供輔助初審，最終判斷仍需由具備資格的消防設備師或相關專業人員確認。
